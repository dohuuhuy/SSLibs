# SSlibs

> Tiêu chí nhẹ và đẹp, có thể cài đặt lại theo yêu cầu ...

## Install npm

```bash
npm i sslibs
```

## Install yarn

```bash
yarn add sslibs
```

## Usage

```tsx
import React, { Component } from 'react'
import { SSCalendar, SSCountDownUI } from 'sslibs'
import 'sslibs/dist/index.css'

const TestLibs = () => {
  return (
    <>
      <SSCountDownUI />
      <SSCalendar />
    </>
  )
}

export default TestLibs
```
